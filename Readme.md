# InstaPhish - Red Team Campaign Simulator

**For authorized penetration testing and security assessments only.**

InstaPhish is a red team tool designed to simulate real-world credential harvesting
campaigns against Instagram login portals. It enables security professionals to
assess organizational susceptibility to phishing attacks.

## Features
- Real-time login page replication
- Session token interception for MFA bypass demonstration
- Live campaign dashboard with credential logging
- Serveo integration for rapid deployment
